Bulk Upload Requirements

  • Upload a FeatureCollection with multiple enumeration areas
  • Each feature must have properties: subAdministrativeZoneIds (array), name, areaCode, description
  • Projection: WGS84 (EPSG:4326)
  • Supported geometry types: Polygon, MultiPolygon
  • Existing areas (same areaCode + subAdministrativeZoneIds combination) will be skipped

Context Filter Active

Only enumeration areas for Sub-Administrative Zone ID: {{ subAdminZoneId }} will be processed.
