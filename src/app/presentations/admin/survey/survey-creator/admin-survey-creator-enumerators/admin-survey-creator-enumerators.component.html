CSV Format Instructions

Upload a CSV file with enumerator information. The system will automatically create new users or use existing ones based on CID.

Required Columns:

  • name - Full name
  • cid - 11-digit CID

Optional Columns:

  • emailAddress - Email
  • phoneNumber - 8-digit phone
  • password - User password

Missing fields will auto-generate: email as {{ "{cid}" }}dummy.nsb.gov.bt, password as CID.
